NEFCHEF - ΟΔΗΓΙΕΣ ΧΡΗΣΗΣ
________________________

Με το πρόγραμμα αυτό μπορείτε να καταγράφετε τις συνταγές σας, ενώ μετριούνται αυτόματα οι θερμίδες της συνταγής, η μάζα και οι θερμίδες ανά 100 γραμμάρια.

Στην αρχική οθόνη θα βρείτε τα δύο βασικά κουμπιά για δημιουργία νέας συνταγής, καθώς και για άνοιγμα συνταγών που έχετε αποθηκεύσει.

Στο πρόγραμμα έχουν ήδη αποθηκευτεί και είναι έτοιμα για προσθήκη σε συνταγή αρκετά συστατικά.
Μπορείτε εύκολα να δημιουργείτε και τα δικά σας συστατικά, στην καρτέλα του συντάκτη υλικών, τα οποία αποθηκεύονται αυτόματα για μελλοντική χρήση.
Στην ίδια καρτέλα μπορείτε να κάνετε επεξεργασία των συστατικών που έχετε επιλέξει για τη συνταγή σας, να προσθέσετε νέα, είτε δικά σας (κουμπί 'προσθήκη νέου συστατικού), είτε αποθηκευμένα ('προσθήκη υλικού από λίστα), να αφαιρέσετε, ή να αλλάξετε τη σειρά εμφάνισής τους.

Στην καρτέλα του συνταγογράφου μπορείτε να ορίσετε τα γραμμάρια του κάθε συστατικού για τη συνταγή σας, καθώς και να γράψετε τις οδηγίες εκτέλεσης της συνταγής.

Με τα κουμπιά της αποθήκευσης μπορείτε να αποθηκεύετε τις συνταγές σας σε αρχεία .rcp.
Μπορείτε επίσης να εξάγετε τη συνταγή σας σε αρχείο .pdf, στο οποίο θα συμπεριλαμβάνονται αυτόματα τόσο τα συστατικά σας, όσο και τα γραμμάρια που έχετε ορίσει, οι οδηγίες εκτέλεσης και οι υπολογισμένες θερμίδες.
Το όνομα του αρχείου της συνταγής θα εμφανίζεται ως τίτλος στο .pdf.
